# WOAION / governance

Repositorio de gobernanza del ecosistema WOAION. Contiene las normas, matrices de vocabulario y plantillas que rigen toda la documentación producida dentro del sistema, tanto interna como pública.

**Este repositorio es la fuente de verdad del sistema documental.**

## Cómo usar este repositorio

### Si sos nuevo en WOAION
1. Leé el [UDS v4.0](./00-normas/00_WOAION_UDS_sistema_unificado.md) para entender la arquitectura del sistema
2. Revisá el [Registro de categorías](./01-matrices/01_REGISTRO_CATEGORIAS.md) para conocer los códigos disponibles
3. Elegí la plantilla correcta en `02-plantillas/` antes de crear cualquier documento

### Si vas a contribuir
Leé el [CONTRIBUTING.md](./CONTRIBUTING.md) para entender el proceso de contribución y las restricciones sobre normas GLB.
